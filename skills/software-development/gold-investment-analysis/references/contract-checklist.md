# 黄金周报契约检查清单（8项必修）

> 来源：2026.7.21 Opus 契约审查确立。缺失任一项 = 契约不合格。

| # | 检查项 | 说明 |
|:--|:--|:--|
| ① | USDCNH + CNH-CNY基差 | D4.2中标注CNH≠CNY超过0.3%时的基差 |
| ② | 统一时点快照 | D4所有价格取SGE收盘时点（约15:00北京时间） |
| ③ | 评分公式+七因子权重 | D2中明确 `score = (composite+2)/4×100`，列出权重 |
| ④ | 模型公允价值来源 | 注明 scorer.py 估值模块（TIPS+DXY回归） |
| ⑤ | 定投建议去产品化 | 禁止任何ETF/基金/股票代码或名称 |
| ⑥ | D4表含判定阈值 | D4.1-D4.6每项有阈值（±0.8%, ±1%, ±10%等） |
| ⑦ | 双审3轮终止条件 | 同一审查层3轮未PASS→停下告知用户 |
| ⑧ | 补发标注+投递确认 | 报告开头「📅 补发（原定日期）」+ 投递成功=完成条件 |

## 其他契约质量要点

- D4校验表统一用4列格式：`| 子项 | 内容 | 结果 | 判定 |`
- 强制术语表：「境内外价差」「公允价值偏离」禁止混用
- 免责声明必须位于**报告末尾**，措辞：「⚠️ 以上分析仅供参考，不构成投资建议。投资有风险，决策须谨慎。」
- 操作建议禁止单一目标价和止盈止损点位
- D3须引用D2至少3个分项评分
- 价格区间标注[推断]标签并给出估算方法
