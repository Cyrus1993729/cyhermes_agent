# DCA 退休金计算器：方法与实践

> 定投（Dollar-Cost Averaging）终值计算的方法论，含多情景对比、费率调整、多阶段策略分析。适用于纳指定投、标普定投等长周期退休规划场景。

## 核心公式

### 日定投终值（按交易日复利）

```python
daily_rate = (1 + annual_rate) ** (1 / trading_days_per_year) - 1
total_days = trading_days_per_year * years
fv = daily_amount * ((1 + daily_rate) ** total_days - 1) / daily_rate
```

**参数约定：**
- `trading_days_per_year = 252`（美股交易日，用于复利计算）
- `workdays_per_year = 250`（中国工作日，用于投入本金计算）
- 复利用252天（资产在美股交易日增值），投入用250天（中国场外基金按工作日扣款）

### 场外QDII基金费率调整

QDII基金管理费+托管费合计约 1.5%/年：
```python
net_rate = gross_rate - 0.015  # 费前 → 费后
```

标普500长期费后中性预期：**8.5-10%**

## 多阶段策略分析框架

### 模式一：单一资产定投+持有

```
阶段1（Y1-Y20）：每日定投资产A
阶段2（Y21-Y30）：停止定投，纯持有资产A
```

终值 = 阶段1定投终值 × (1 + r)^10

### 模式二：资产切换（纳指→标普）

```
阶段1（Y1-Y20）：每日定投纳指
阶段2（Y21）：全部转标普500
阶段3（Y21-Y30）：纯持有标普（或继续定投标普）
```

终值 = 纳指定投终值 × (1 + r_sp)^10 [+ 标普新增定投]

### 模式三：纯定投不停

```
阶段1（Y1-Y30）：每日定投同一资产
```

终值 = daily × ((1+r)^n − 1) / r

## 财务承受力分析框架

### 步骤1：计算年结余

```
年结余 = 税后收入 − 日常开销 − 保险 − 其他固定支出
```

### 步骤2：定投金额占结余比例

| 占比 | 评价 | 适用条件 |
|------|------|----------|
| <40% | 保守 | 收入不稳定、有大额支出计划 |
| 40-60% | **适中** | 收入稳定、已留紧急备用金 |
| 60-80% | 积极 | 高收入、无负债、无大额计划 |
| >80% | 过激 | 几乎无缓冲，意外支出会打断定投 |

### 步骤3：压力测试

假设日常开销膨胀20%，重新计算受压结余下的定投占比：
- 受压占比 <70%：安全
- 受压占比 70-85%：警戒
- 受压占比 >85%：危险，建议调低定投额

### 步骤4：边界条件调节

| 条件 | 对定投金额的影响 |
|------|-----------------|
| 无子女 | +10-15%结余可用于投资（无需教育基金） |
| 有房贷 | −5-10%（月供占用现金流） |
| 双收入家庭 | +5-10%（风险共担） |
| 父母需赡养 | −5-10%（预留赡养费） |
| 职业不稳定 | −10-15%（需更大现金缓冲） |

## 2026-06-29 实践案例

用户：33岁男性，无子女，年结余¥9万，纳指定投规划。

**分析结果：**
- ¥200/日 = ¥5万/年 = 56%结余 → **甜蜜点**
- 中性预期：20年后¥321万（投¥100万），30年后¥959万（投¥150万）
- 纳指20年+持有10年（停止定投）：¥872万（仅投¥100万，省¥50万现金流）
- 纳指20年+转标普持有10年：¥833万（牺牲¥39万收益换降低波动）
