# 产业链早信号采集与交叉验证方法论

> 2026-06-25 与 Claude Opus 三轮咨询成果

## 一、供应链时间机器：信号分层

上游物理承诺必须先于下游发生，在公开信息中留下时间错开的痕迹：

```
论文/标准/专利（1-3年，脏）
  → 招聘/私有融资（6-18月）
    → 环评/能评/项目备案（9-18月，变硬）
      → 设备招标/订单（6-12月）
        → 海关到货（3-6月）
          → 月营收/稼动率（数周-1季）
            → 财报/股价（0领先，已定价）
```

核心规律：清晰度和领先时间反向。最早信号最不可信。真正的 edge 是"多个早信号交叉确认"。

## 二、信号角色分工

| 角色 | 信号类型 | 特点 |
|------|---------|------|
| 触发器（雷达层） | 待定 | 广而早，告诉你"往哪看"。不要用招聘JD（假阳性太高） |
| 确认器 | 设备招标、环评备案、财报在建工程 | 准而硬，确认"这是真的" |

## 三、三层交叉确认

每个疑似信号回答三个正交问题：

| 问题 | 查什么 |
|------|--------|
| 是不是真投钱了？ | 设备招标/订单（关键设备） |
| 是不是量产级？ | 环评里的产能数字、洁净室面积 |
| 是不是被需求拉动？ | 下游同期动作（合作公告、同类招聘） |

三条都命中 = 高置信度。时间序列自洽性是最强反噪声检验。

## 四、信息源按 alpha 分级

- **有 API/结构化（低 alpha）**：SEC EDGAR、Google Patents BigQuery、Tushare/AkShare
- **需爬虫（高 alpha，重点投入）**：环评/能评/项目备案（省市县三级，格式不统一）、招投标平台
- **只能人工判断**：客户认证真实性、管理层话术可信度、技术路线判断

## 六、先进封装"必留痕"节点

因重资产+化学工艺（电镀/RDL/TSV/显影），在以下渠道几乎隐身不了：
1. 环评公示（含工艺、产能数字、设备清单、化学品用量）
2. 节能审查（能评）
3. 固定资产投资项目备案（发改委）
4. 上市公司超阈值资本开支（强制披露）
5. 设备进口海关报关（HS code 可追）
