# 技术环境配置

## 代理
- Clash Verge 7897 规则模式
- Claude Code 需 HTTP_PROXY/HTTPS_PROXY=127.0.0.1:7897
- Python 主程序不设代理，由 Clash 路由
- Sina API 直连，yfinance/FRED 需走代理

## 模型
- 默认: deepseek-v4-pro (provider: deepseek)
- Kimi vision: provider=moonshot-cn, KIMI_CN_API_KEY=KIMI_API_KEY
- 中国 key 仅 api.moonshot.cn 有效，.ai 不行

## 关键工具
- HTML→PNG 微信分享：Playwright + 系统 Edge (channel=msedge)，无需装 Chromium
- ffmpeg 通过 imageio-ffmpeg 获取路径
- DeepSeek API key: ~/deepseek_key.txt

## 当前版本
- Hermes v0.17.0 (2026-06-21 升级自 0.14.0)
- Python 3.11.15（3.12 升级被 git checkout 回退，遇 #42810 bug 再修）
- 桌面客户端已装

## 绝对禁止
- taskkill /F /IM python.exe → 杀死 Agent 自身
- 杀进程必须 taskkill /PID <pid>
- 资源排查用 ps aux 或任务管理器
