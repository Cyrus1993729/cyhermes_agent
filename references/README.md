| 目录 | 用途 | 示例 |
|:---|:---|:---|
| 投资分析/ | 黄金、美股等投资研究记录 | 金价分析报告、美股估值对比 |

## 使用方式

每完成一次有价值的研究或任务，Hermes 会自动将结论存入对应目录。
每月一次成长对话时，Hermes 会扫描此目录来评估进展。
