## v0.18.0 核心更新摘要

### 1. MoA 进化为第一公民
- 不再需要 `/moa china` 命令，MoA 预设作为可选模型出现在模型选择器里
- 每个参考模型的输出分块展示（能看到 GPT/Claude/Grok 各自的分析）
- 汇总器结果流式输出（不等结果一次性返回）

### 2. 省钱：自进化走便宜模型
- post-turn 的记忆/skill 学习（判断是否该存 memory 或写 skill）现在走辅助模型而非主模型
- 用户刚搭的 lessons.md 闭环和 L1 审查系统 = 这条直接省钱

### 3. `/learn` — 描述就能创建 skill
- 打 `/learn 帮我分析小红书帖子`，Hermes 自动蒸馏成 skill
- 用户有 30+ 个 skill，简化 skill 创建流程

### 4. `/journey` — 记忆时间线
- 可以看到 Hermes 学了什么、什么时候学的
- 桌面端还有可视化记忆图谱
- 用户是 WeChat 用户，桌面端用不上但时间线可能有用

### 5. 后台并行 delegation
- delegate_task 的子 agent 可以后台跑，不堵对话

### 6. 安全加固
- MCP 配置持久化攻击面锁定

- aiohttp CVE  floors

### 7. Agent 自验证（跟我们已有的重叠）
- Hermes 记录验证证据
- `/goal` 完成合约
- 但用户已经自己搭了 L1 千问审查层 + sprint-contract

### 8. P0/P1 清零
- 496 P0/P1 issues + 196 P0/P1 PRs 全部关闭
- 稳定性比 v0.17.0 更好
